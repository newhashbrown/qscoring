# QScoring

Instant quant scores for any stock. Enter a ticker, get a data-driven Quant Score and a clear buy / hold / short signal.

Site: https://qscoring.com

## Contents

- `index.html` — landing page
- `QScoring_Business_Plan.docx` — business plan

## Local preview

Open `index.html` directly in a browser, or serve the folder:

```bash
python -m http.server 8000
# then visit http://localhost:8000
```
